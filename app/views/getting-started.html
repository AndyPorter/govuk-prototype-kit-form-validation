Getting started

Adding the validation code to the kit

1. Install the library

Open terminal navigate to your repository

Enter the following code to install the validate.js library

npm install validate.js

2. Set up routes

With the prototype open your text editor, go to the app folder and the routes.js file

Under the 'Add your routes here' comment, add the following code:

router.use('/node_modules', express.static('node_modules'))

3. Adding scripts

Now go to the views/includes folder and open the scripts.html file. Add the following underneath the existing <script> src="/public/javascripts/jquery-1.11.3.js" </script>

<script> src="/node_modules/validate.js/validate.min.js" </script>
<script> src="/public/javascripts/custom-validator.js" </script>
<script> src="/public/javascripts/form-validation.js" </script>
