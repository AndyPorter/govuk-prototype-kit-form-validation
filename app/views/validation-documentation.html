Adding validation rules to an imput field

There are various types of validation you can do - for example, to ensure the field isn't blank, if a checkbox or radio button has been selected, if it's all numbers, if it's a pattern like a postcode

If you want to validate or constrain how much text can be inputted - there are other standard methods to do this, such as setting a field length attribute (minlength or maxlength, or using the character count govuk code for the text area.
